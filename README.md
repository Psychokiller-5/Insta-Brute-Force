 ## 
  
 ### Installation

- Linux
  ```
  git clone https://github.com/SamueleAmato/Instagram-Brute-force-2022/
  cd Instagram-Brute-force-2022
  cd Linux
  cd setup
  python3 run_setup.py
  cd ..
  python3 main.py
  ```

- Termux
  ```
  git clone https://github.com/SamueleAmato/Instagram-Brute-force-2022/
  cd Instagram-Brute-force-2022
  cd Termux
  pip3 install -r requirements.txt
  python3 main.py
  ``` 
- Windows
  ```
  Download zip file
  Go inside Windows folder
  Run main.py or main.exe
  
  ```
