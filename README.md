 ## 
  
 ### Installation

- Linux
  ```
  git clone https://github.com/Psychokiller-5/Insta-Brute-Force
  cd Insta-Brute-Force 
  cd Linux
  cd setup
  python3 run_setup.py
  cd ..
  python3 instabf.py
  ```

- Termux
  ```
  git clone https://github.com/Psychokiller-5/Insta-Brute-Force
  cd Insta-Brute-Force 
  cd Termux
  pip3 install -r requirements.txt
  python3 instabf.py
  ```
